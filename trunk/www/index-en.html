Veggente

Veggente project is a thesis project in eHealth area at Politecnico of Milano

Targets

Two are the targets in this project:
  • To allow building of HL7 v.3 messages from data fetched from structurally different HL7 v.3 messages. (v. architecture).
  • The Secondary target is to extend input data inference allowing to import data coming from messages conforming to other standards, using sintactic and semantic mappings. (v. complete architecture)
    • Here it is possible to find some more information.

    Status

    Ontologies

    HL7 RIM, Datatype, CMETs and HMDs have been converted into OWL ontologies. There's still an ongoing work to check if all information have been correctly treated. These ontologies can be found here.
    Input HL7 data is in the SubVersion repository.

    Repository

    The RDF/OWL repository is based on Redland and exports its functions using SOAP. It is a glue between storage and parsing capabilities (offered by Redland) and the inference engine.

    Inference

    CWM has been chosen as the inference engine and modified to allow it to fetch data coming from Redland. Currently SQLite and MySQL backends are supported, and the next target is to give CWM access to Redland's in-memory model.
